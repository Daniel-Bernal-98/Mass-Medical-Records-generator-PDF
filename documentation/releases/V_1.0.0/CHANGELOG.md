## [1.0.0] - Initial Release

### Added

* Medical Records package generation from claim CSV files.
* Individual claim package generation.
* Bulk package generation.
* Automatic Table of Contents generation.
* SOAP note matching using DOS and CPT information.
* OCR-based payer letter identification and matching.
* PDF compression and size reduction utilities.
* Automatic cover letter population.
* PDF flattening and form field processing.
* Processing progress tracking and execution logs.
* Portable Windows executable support.
* User-friendly graphical interface built with Tkinter.
